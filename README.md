# Cotiara

## Работа с переменными

### Объявление переменных:
Создает переменную с указанным начальным значением.

```cotiara
var a = 1
```

### Изменение значения переменной:
Присваивает переменной новое значение.

```cotiara
a = 2
```

## Арифметические операции

### Сложение (+)
Складывает два числа.

```cotiara
var sum = 2 + 3 /* 5 */
```

### Вычитание (-)
Вычитает второе число из первого.

```cotiara
var difference = 5 - 2 /* 3 */
```

### Умножение (*)
Умножает два числа.

```cotiara
var product = 2 * 3 /* 6 */
```

### Деление (/)
Делит первое число на второе.

```cotiara
var quotient = 6 / 2 /* 3 */
```

## Логические операции

### Равенство (==)
Проверяет, равны ли два значения.

```cotiara
var isEqual = 1 == 1 /* True */
```

### Неравенство (!=)
Проверяет, различаются ли два значения.

```cotiara
var isNotEqual = 1 != 2 /* True */
```

### Больше (>)
Проверяет, больше ли первое значение, чем второе.

```cotiara
var isGreater = 2 > 1 /* True */
```

### Меньше (<)
Проверяет, меньше ли первое значение, чем второе.

```cotiara
var isLess = 1 < 2 /* True */
```

### Больше или равно (>=)
Проверяет, больше или равно ли первое значение второму.

```cotiara
var isGreaterOrEqual = 1 >= 1 /* True */
```

### Меньше или равно (<=)
Проверяет, меньше или равно ли первое значение второму.

```cotiara
var isLessOrEqual = 1 <= 2 /* True */
```

### Логическое И (&&)
Возвращает True, если оба выражения истинны.

```cotiara
var a = 1 == 1
var b = 2 > 1
var bothTrue = a && b /* True */
```

### Логическое ИЛИ (||)
Возвращает True, если хотя бы одно из выражений истинно.

```cotiara
var a = 1 == 1
var b = 1 > 2
var eitherTrue = a || b /* True */
```


## Вывод текста на экран

### echo
Выводит заданный текст или значение переменной. Текст указывается без кавычек.

```cotiara
echo Привет, мир! /* Выводит текст: Привет, мир! */
```

```cotiara
var a = 123
echo %a% /* Выводит число: 123 */
```

## Комментарии

Используются для добавления пояснений в код, не влияют на его выполнение.

```cotiara
/* Это многострочный комментарий */
```

## Циклы

### loop
Повторяет вложенный код указанное количество раз.

```cotiara
loop 5 {
    echo hi
}
/* Выводит:
hi
hi
hi
hi
hi
*/
```

## Условные конструкции

### if
Выполняет код внутри блока, если условие истинно.

```cotiara
if 1 == 1 {
    echo hi
}
/* Выводит: hi */
```

## Функции

Функции позволяют группировать код для многократного использования.

### func
#### Объявление функции
Определяет функцию с указанными параметрами.

```cotiara
func sum (a, b) {
    var c = %a% + %b%
    echo %c%
}
```

### call
#### Вызов функции
Вызывает ранее определенную функцию с указанными аргументами.

```cotiara
call sum(2, 2) 
/* Выводит: 4 */
```

## Событие нажатия клавиши

### on_press
Функция с двумя аргументами:
- Первый аргумент отображает состояние клавиши: нажата (0) или отпущена (1).
- Второй аргумент — десятичный скан-код клавиши, определённый по таблице.

```cotirara
func on_press(state, key) {
    if state == 0 {
        echo "Клавиша с кодом %key% нажата"   
    } else if state == 1 {
        echo "Клавиша с кодом %key% отпущена"
    }
}
```
